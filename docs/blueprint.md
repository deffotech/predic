# **App Name**: VoteMapper

## Core Features:

- Home Page: Landing page with 'Enter' (contact form) and 'Add' (map) buttons.
- Contact Form: Collects user details like age, party, name, etc.
- Geolocation Capture: Automatically capture latitude and longitude when adding information on the map.
- Interactive Map: Display collected data points on a map using Leaflet, including options to add, edit, view, and remove data with confirmation.
- Data Export: Enable data export to Excel format.
- Admin Dashboard: Visualize the voting preferences based on the collected data and predict the maximum number of people prepared to vote for a particular party.
- Color-Coded Leaflets: Display each political party on the map using its designated color: saffron, red, black, white, yellow, and green.

## Style Guidelines:

- Primary color: Dark purple (#301934), echoing the sophisticated but technological look in the reference image.
- Background color: Very dark grayish purple (#1A131D) -- nearly black but still of the same hue as the primary.
- Accent color: Electric blue (#7DF9FF) for highlights and interactive elements; an analogous hue with a brightness and saturation contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
- Use clean, modern icons for map markers and UI elements.
- Responsive design for optimal viewing on mobile and tablet devices.
- Subtle animations on button interactions and data updates.
